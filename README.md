# Semantic Sensing Communication using SDR and AI-Assisted Processing

Final-year Electronic Engineering with Space Systems dissertation project focused on low-latency wireless image and video transmission using software-defined radio hardware and semantic-inspired preprocessing techniques.

The project was developed using MATLAB, Simulink and ADALM-Pluto SDR hardware, with testing carried out using both simulation and real over-the-air wireless transmission.

---

## Project Overview

The project was developed in several stages, starting with basic QPSK communication before progressing towards OFDM image transmission, SDR hardware validation and low-latency wireless video streaming.

The final stage explored a lightweight semantic-inspired hybrid compression prototype which investigated how preprocessing techniques could be integrated into the communication workflow to prioritise visually important information before transmission.

## Results

The final SDR system achieved stable low-latency wireless video transmission under indoor line-of-sight conditions.

The semantic-inspired hybrid prototype demonstrated how lightweight preprocessing and packetisation techniques could be integrated into the SDR workflow for future semantic communication research.
